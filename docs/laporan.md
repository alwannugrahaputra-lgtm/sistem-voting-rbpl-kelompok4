# Ringkasan Praktikum Git RBPL

Repository ini dibuat untuk memenuhi tugas praktikum Git pada mata kuliah RBPL.

## Fitur
- Tambah kandidat
- Voting
- Statistik dan persentase
- Tentukan pemenang
- Penyimpanan data JSON
- Unit test

## Branch yang Digunakan
- fitur-core-system (Alwan)
- fitur-statistik (Alwan)
- fitur-dokumentasi-farid
- fitur-validasi-adinda
- fitur-unit-test-syahlaa

## Pengujian
Unit test dijalankan menggunakan:
python -m unittest discover tests

Semua test berhasil.
