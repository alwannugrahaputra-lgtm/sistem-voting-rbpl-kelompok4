# Sistem Voting Online CLI - RBPL

## Deskripsi
Aplikasi voting sederhana berbasis CLI menggunakan Python.
Project ini dibuat untuk praktikum Git mata kuliah RBPL.

## Fitur
- Tambah kandidat
- Voting kandidat
- Menampilkan statistik dan persentase suara
- Menentukan pemenang
- Penyimpanan data ke JSON

## Cara Menjalankan
1. Masuk ke folder src
2. Jalankan:
   python main.py

## Struktur Folder
- docs/ : dokumentasi
- src/ : source code
- tests/ : unit testing
- README.md : dokumentasi project

## Branching Strategy
- main → branch final
- development → penggabungan fitur
- fitur-core-system → fitur dasar
- fitur-statistik → fitur statistik
